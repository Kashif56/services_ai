Booking Widget Example

This page demonstrates how to embed the booking widget into any website.

Embeddable Responsive Customizable

ðŸ“‹ How to Use

To embed the booking widget on your website, add the following code:

<!-- Booking Widget Container -->
<div id="booking-widget"
data-business-id="your-business-id"
data-api-url="{{ request.scheme }}://{{ request.get_host }}"
data-primary-color="#8b5cf6">
</div>

<!-- Widget Loader Script -->
<script src="{{ request.scheme }}://{{ request.get_host }}/static/js/booking-widget-loader.js"></script>

Configuration Options

  • data-business-id (required): Your business ID identifier
  • data-api-url (optional): The base URL of your API (defaults to current domain)
  • data-primary-color (optional): Custom primary color in hex format (defaults to #8b5cf6)

ðŸ”Œ API Endpoints

The widget uses the following public API endpoints:

  • GET /bookings/widget/{business_id}/config/ - Get widget configuration
  • GET /bookings/widget/{business_id}/service-items/{service_id}/ - Get service items
  • GET /bookings/widget/{business_id}/check-availability/ - Check staff availability
  • POST /bookings/widget/{business_id}/create/ - Create booking
